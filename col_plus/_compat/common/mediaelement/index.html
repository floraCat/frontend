官网：http://mediaelementjs.com/


调用：
	html:
		<video src="media/echo-hereweare.mp4"></video>

	js:
		new MediaElement('player1', {
			success: function(video) {
				video.play();
			}
		});

参数：
	success回调函数的video参数有如下方法：
		load
			加载视频

		play
			播放视频
			
		pause
			暂停视频
